## рџ“Њ About The Project (Budget 2026-27: )

**Pak Tax Calculator** is a modern, responsive, and production-ready Pakistan Income Tax Calculator built with **React 19**, **Vite**, and **Tailwind CSS**. It provides accurate tax calculations based on official **FBR (Federal Board of Revenue)** tax slabs for the tax years **2025-26** and **2026-27**.

This platform helps Pakistani taxpayers calculate their **Salary Tax**, **Business Tax**, **Rental Tax**, **Capital Gain Tax**, and **Agriculture Tax** instantly вЂ” all in one place with a beautiful and intuitive interface.

## рџљЂ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/YasirAwan4831/pak-tax-calculator.git
```

### 2. Navigate to Project Directory

```bash
cd pak-tax-calculator
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```

### 5. Build for Production

```bash
npm run build
```

### 6. Preview Production Build

```bash
npm run preview
```
